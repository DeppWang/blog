必备

  • Typora: 简洁、好用、所见即所写的 Markdown 客户端编辑器
  • 奇妙清单：基于云端工作的任务管理（to-do-list）应用软件
  • SSR： 科学上网软件，中文教程，自建参考 耗子叔教程
  • LastPass：全平台同步密码，iOS 版美区 id 可下载。也可用 BitWarden，开源免费，中文教程

macOS

  • Alfred：针对 macOS 的效率启动器，具有网页搜索、翻译、打开网址/软件/浏览器书签等功能。可使用 Workflows 扩展功能，Workflows 收费，中文教程
  • Manico: 可自定义快捷键切换和打开应用程序的工具，相当于 Windows 的「Win键+数字」。建议给应用自定义快捷键。默认免费，去除收费提醒 25 元
  • Popclip: 可直接操作任何页面选中的字符，直接实现搜索、复制、剪贴、删除、翻译、发音等功能，可添加丰富的插件。官网下载可试用 182 次，收费 88
  • Dash: 可以让你离线访问所有主流语言 API 的软件，有 iOS 版本
  • 休息一下：定时黑屏提醒休息的工具
  • iPic：针对 macOS 的图床神器，可结合 Typora 使用，复制图片到 Typora 自动上传到图床，图床可自定义，使用自定义图床收费，建议开启图片压缩功能，可加快访问速度，减少流量（如果使用自定义图床）。中文教程
  • iPicMover：iPic 作者开发的博客图片迁移工具
  • Parallels Desktop: macOS 上最好用的虚拟机，同时运行 Windows 和 macOS 不是梦。收费，Parallels Desktop 13 破解视频教程（YouTube），备用下载地址（Google Drive）
  • NeatDownloadManager: 针对 macOS 的免费下载工具，也有 Windows 版，并发数设为 32 可提速
  • Bartender 3：一个管理「人满为患」的顶部菜单栏图标的工具，可免费使用 4 周，后续收费 112 左右
  • Dozer：相当于免费版 Bartender
  • Xnip：一个可在 macOS 上长截屏的免费小工具
  • Pock：一个让 Touch Bar 「变废为宝」（显示 Dock）的免费小工具。GitHub 地址

Windows

  • Wox：针对 Windows 的效率启动器，相当于 macOS 的 Alfred。GitHub 地址、中文教程
  • Everthing：基于名称快速定位文件和文件夹的软件，可配合 Wox 使用
  • 火绒：轻量、无广告电脑安全软件，相比 360、腾讯，体验更好
  • 分流抢票：集预约和捡漏的免费 Windows 客户端抢票软件，无套路，不用加速包，抢票率高。需要一直后台运行，可以配合 TeamViewer 远程操控。12306 出了官方「抢票」功能 — 候补购票
  • Windows Terminal：微软全新命令行工具。教程、GitHub 地址

iOS

  • PPHub：一个漂亮的 iOS GitHub 客户端，收费 18 元
  • Pretext：iOS 上的 MorkDown 编辑器，Pretext + iCloud 云盘（或 Dropbox） + Typora，可实现笔记多端实时同步，详见 Typora 官方教程，免费
  • Working Copy：一个实用的 iOS Git 客户端， push 功能收费，有时候会打折（原价 108）
  • VOA：听 VOA 慢速/快速英语的 iOS 客户端，免费
  • 腾讯手机管家：电话短信防骚扰 App，免费
  • 2 岁学说话：「听音识图」的英语学习 APP

Android

  • 健身宝典：一款无广告、动作全的器械健身 App，有动画动作指导、锻炼计划设置、锻炼数据记录等功能，唯一缺点是不能同步到云
  • My GitHub：一款查看个人 GitHub 的 App
  • Top GitHub：一款查看 GitHub 热门项目（Trending）的 Android 客户端
  • VOA 慢速英语：一款可语音测评的 VOA 慢速英语 Android App。另有VOA常速英语
  • Auto.js: Android 平台上不需要 Root 权限的 JavaScript 自动化软件，可用于开发自动收取蚂蚁森林能量等操作。GitHub 地址
  • AIDE：一款支持在 Android 上运行代码的 App（默认是一个安卓项目）

Alfred3 Workflows

  • <译> Alfred 官网 的 Workflows 帮助
  • alfred-github-workflow：提供查看个人 GitHub 仓库、stars，搜索用户、仓库等一系列功能
  • alfred-emoji-workflow：查看常用的 emoji
  • alfred-clipboard-ocr：截屏后 Alfred 输入关键字，自动识别文字，并复制到剪贴板
  • Dash：快捷查询 Dash API
  • alfred-stackoverflow：不打开网页搜索 Stack Overflow，并得到结果
  • 有道翻译加强版：集翻译、发音、添加单词本、复制剪贴板功能一体
  • V2EX：获取 V2EX 最新、最热文章
  • ip-address：查看本地 ip、外网 ip
  • Workflow Sercher：搜索下载 alfredworkflow.com 上的 Workflow

PopClip Extensions

  • PopClip Extensions 官网
  • Google-Translate：查询更快的翻译插件

Chrome 插件

  • Save to Pocket：将待阅读的文章加入到 Pocket 中，Pocket 是一个全平台阅读软件
  • 彩云小译：双语对照网页翻译插件
  • Octotree：将 GitHub 项目代码以目录的层次展示。GitHub 地址
  • Tampermonkey ：俗称油猴，一个用户脚本管理器，脚本安装地址 — Greasy Fork。中文教程
  • Adblock Plus ：广告拦截插件
  • LastPass ：LastPass 插件，自动填充账号密码
  • 为什么你们就是不能加个空格呢？：自动在网页中所有的中文字和半形的英文、数字、符号之间插入空白的插件。GitHub 地址
  • Dark Reader：黑色主题，适用于任何网站，完全免费。GitHub 地址
  • RSS Feed Reader：在工具栏查看订阅的 RSS 或 Atom feeds 文章
  • Free Download Manager ：免费的，支持 BT 磁力下载的插件，用于唤起 客户端 下载。客户端有 Windows 和 macOS。中文教程
  • OneTab ：将 Chrome 打开的浏览器标签合成一个，减少浏览器内存。中文教程
  • Google 翻译：翻译浏览网页文字。中文教程
  • Ghostery：自动学习去阻止不可见的追踪器
  • JSON Formatter：格式化 URL 接口数据为 JSON 格式的插件

油猴脚本

  • 脚本安装地址 — Greasy Fork。
  • 破解 VIP 会员视频集合：一款破解各大视频网站 VIP 视频的脚本
  • AC-baidu：一款去除（百度）搜索引擎右侧广告，更改搜索页样式的脚本
  • 百度网盘直链下载助手：一款生成 BT 链接的的百度网盘下载助手

写作

  • Tyropa: 简洁、好用、所见即所写的 Markdown 客户端编辑器
  • 有道云笔记：支持 MarkDown，实时同步笔记的多客户端软件
  • chinese-copywriting-guidelines：介绍一些中文文档排版时的规范和技巧的 wiki
  • OpenWrite: 将文章一键分发到多个科技平台上的工具，中文教程
  • Markdown Nice：美观的公众号在线排版工具
  • Behance：主要是各种插画创意图片的图片素材网站
  • Pexels：分享免费图片和视频的图片素材网站
  • FlipaClip：一款绘制动画的软件，通过一帧帧的图像最后合成动画。中文教程
  • Processon：一个支持流程图、思维导图、原型图、UML、网络拓扑图、组织结构图等的在线画图网站
  • draw.io：跟 Processon 类似
  • SMMS：一个免费的在线图床网站，现可以管理已上传的图片（2019.8.3）
  • PicGo：图片上传和管理的图床客户端，支持 macOS、Windows、Linux。GitHub 地址
  • carbon: 将代码块转换为图片的网站
  • themostdangerouswritingapp：一个丧心病狂的写作软件，停就删稿！GitHub 地址
  • Graphviz：一个利用脚本生成各种类型图表的软件。中文教程–>

PC App

  • 网易 Mumu：网易出品的安卓模拟器

App

  • 爱玩吉他：一个通过游戏解锁进阶学习吉他的 App，也有大量吉他谱
  • Snapseed：一款 Google 出品的免费修图 App，功能强大，容易上手
  • 翻译：一款 Google 出品的翻译 App，比较准，Android 版悬浮窗很实用，缺点是有的单词没音标
  • Rosetta Stone：主要以「听音辨图」的自然方法学习英语（或其他外语）的软件，收费挺贵，Android 破解版 看这里（提取码: um3h），中文教程

小程序

  • 彩云天气：比较准的简洁的查询天气的小程序，网页端
  • Gitter：GitHub 小程序客户端，目前感觉广告有点多，功能还有待完善（2019.7.27）。GitHub 地址
  • 健身宝典 Lite：简洁的器械健身动画动作指导的小程序

阅读

  • 鸠摩搜书：一个能搜索全网电子书的网站
  • Medium：一个集众多优秀文章的网站，也是很多技术大牛的写作平台。有 iOS、Android 客户端
  • 西窗烛：一个体会诗词之美的 App

问答

  • wikiHow：一个学习了解常见知识的的问答网站
  • Stack Overflow：一个大牛云集的 IT 技术问答网站，有 Android 客户端
  • Stack Exchange：一个涵盖各个领域的问答网站，包括 Stack Overflow 的问题。有 iOS、Android 客户端

视频

  • 疯狂影视搜索：一个能搜索全网影视资源的网站
  • 人人视频：一个看美剧/韩剧/日剧/泰剧的好 App
  • VLLO：一款功能强大，操作简单的视频剪辑 App
  • VUE：一个精品短视频社区

网站

  • RecordScreen.io：一款只需浏览器便可一键录屏的神器。中文教程
  • FACE YOURMAGA：一个自己 DIY 制造卡通头像的网站
  • I Tell You：一个 Windows 原版软件下载站点
  • Server酱：一款从服务器推报警和日志到手机的工具
  • TeamViewer：一款远程操控的软件，比 QQ 远程操控功能强大
  • PDF 拆分：一个免费拆分 PDF 的网站
