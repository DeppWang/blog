必备

  • Typora - 简洁、好用、所见即所写的 Markdown 客户端编辑器。可以使用 Dropbox + MWeb 实现全平台同步
  • Microsoft To Do - 基于云端工作的任务管理（to-do-list）应用软件，目前没有原来 奇妙清单 好用
  • V2rayU - 科学上网软件，中文教程，自建参考 耗子叔教程
  • LastPass - 全平台同步密码，iOS 版美区 id 可下载，自动填充很实用。也可用 BitWarden，开源免费，中文教程
  • 联通沃助理 - 可漏话提醒、语音留言、智能应答，可微信搜索「沃助理」开通。本质上是利用来电转移功能

macOS

  • Alfred - 针对 macOS 的效率启动器。具有网页搜索、翻译、打开网址/软件/浏览器书签等功能。可使用 Workflows 扩展功能，Workflows 收费，中文教程
  • Manico - 可自定义快捷键切换和打开应用程序的工具，相当于 Windows 的「Win键+数字」。建议给应用自定义快捷键。默认免费，去除收费提醒 25 元
  • HyperSwitch - 窗口切换工具。可使用 Cmd + Tab 代替「三指上划」切换所有应用窗口，可使用 Cmd + ` 切换当前应用窗口，非常实用。如提示不能打开，在「Security & Privacy」中设置一下。
  • Popclip - 可直接操作任何页面选中的字符，直接实现搜索、发音、字典、翻译、复制、剪贴等功能，可添加丰富的插件。官网下载可试用 182 次，收费 88
  • 微信 Mac 版公测版 + WeChatPlugin-MacOS - 可打开小程序；可将烦人群信息「置底」(Sticky Bottom)
  • Dozer - 一个管理「人满为患」的顶部菜单栏图标的开源免费工具
  • Go2Shell - 在 Finder 里快速进入 shell 环境
  • Hazel - 升级版 Automator
  • Xnip - 一个可在 macOS 上长截屏的免费小工具
  • Snip - 可缩小截屏所占内存的截屏软件，「设置」选择「高清屏 ( Retina) 下截取 1X 大小图片」
  • Parallels Desktop - macOS 上最好用的虚拟机，同时运行 Windows 和 macOS 不是梦
  • NeatDownloadManager - 针对 macOS 的免费下载工具，也有 Windows 版，并发数设为 32 可提速
  • Dash - 可以让你离线访问所有主流语言 API 的软件，有 iOS 版本
  • Toolbox - 一个同步 Jetbrains 开发环境配置的工具
  • 休息一下 - 定时黑屏提醒休息的工具

Alfred Workflows

  • <译> Alfred 官网 的 Workflows 帮助
  • alfred-github-workflow - 提供查看个人 GitHub 仓库、stars，搜索用户、仓库等一系列功能
  • alfred-clipboard-ocr - 截屏后 Alfred 输入关键字（ocr），自动识别文字，并复制到剪贴板
  • alfred-google-translate - 基于 Google 翻译 API 的翻译插件，无需打开网页
  • 有道翻译加强版 - 集翻译、发音、添加单词本、复制剪贴板功能一体
  • Alfred-HackerNews - Hacker News Alfred Workflows
  • V2EX - 获取 V2EX 最新、最热文章，直接下载
  • alfred-emoji-workflow - 查看常用的 emoji
  • alfred-pangu-workflow - 排版剪贴板文字，加上合适的空格，如：中文与英文、数字之间加上空格
  • Dash - 快捷查询 Dash API
  • alfred-stackoverflow - 不打开网页搜索 Stack Overflow，并得到结果
  • ip-address - 查看本机内网 ip、外网 ip

Windows

  • Wox - 针对 Windows 的效率启动器，相当于 macOS 的 Alfred。GitHub 地址、中文教程
  • Everthing - 基于名称快速定位文件和文件夹的软件，可配合 Wox 使用
  • 火绒 - 轻量、无广告电脑安全软件，相比 360、腾讯，体验更好
  • 分流抢票 - 集预约和捡漏的免费 Windows 客户端抢票软件，无套路，不用加速包，成功率高。需要一直后台运行，可以配合 TeamViewer 远程操控。12306 官方「抢票」功能 — 候补购票
  • Windows Terminal - 微软全新命令行工具

iOS

  • Shadowrocket - 俗称小火箭，科学上网，收费
  • PPHub - 一个漂亮实用的 iOS GitHub 客户端，收费 18 元
  • Octal - Hacker News 的 iOS 客户端
  • Pin - iOS 上的剪贴板神器
  • 50 音起源 - 设计出色的日语学习 APP
  • MWeb - iOS 上的 MorkDown 编辑器，MWeb + iCloud 云盘（或 Dropbox） + Typora，可实现笔记多端实时同步与编辑，详见 Typora 官方教程，免费
  • 语音报时闹钟 - 支持整点报时、半点报时、自定义任意时刻报时、间隔无限报时、倒计时等等
  • Working Copy - 一个实用的 iOS Git 客户端， push 功能收费，有时候会打折（原价 108）
  • 腾讯手机管家 - 电话短信防骚扰 App，免费

Android

  • 健身宝典 - 一款无广告、动作全的器械健身 App，有动画动作指导、锻炼计划设置、锻炼数据记录等功能，唯一缺点是不能同步到云
  • My GitHub - 一款查看个人 GitHub 的 App
  • Top GitHub - 一款查看 GitHub 热门项目（Trending）的 Android 客户端
  • AIDE - 一款支持在 Android 上运行代码的 App（默认是一个安卓项目）

Browser Extensions

  • NeatDownloadManager - 用于启动 NeatDownloadManager 下载
  • LastPass - LastPass 插件，自动填充账号密码
  • RAINDORP.io - 全平台同步书签
  • Save to Pocket - 将待阅读的文章加入到 Pocket 中，也可将作为一个保存收藏文章的仓库。Pocket 是一个全平台阅读软件
  • RSS Feed Reader - 在工具栏查看订阅的 RSS 或 Atom feeds 文章，有 APP 客户端
  • Adblock Plus - 广告拦截插件
  • Shortkeys - 自定义浏览器没有的快捷键，如关闭其他标签页
  • Old School GitHub - 还原 GitHub 为旧样式，主要更改为居中显示
  • 为什么你们就是不能加个空格呢？ - 自动在网页中所有的中文字和半形的英文、数字、符号之间插入空白的插件
  • Dark Reader - 夜间模式，适用于任何网站，完全免费
  • JSON Formatter - 格式化 URL 接口数据为 JSON 格式的插件
  • Octotree - 将 GitHub 项目代码以目录的层次展示
  • Tampermonkey - 俗称油猴，一个用户脚本管理器，脚本安装地址 — Greasy Fork。中文教程
  • STTF Url Generator - 用于生成打开即可高亮复制内容的链接
  • Grammarly - 浏览器输入框英语输入时语法检测与提示
  • Readlang Web Reader - 类似 Kindle Word-Wise 的英语辅助阅读插件
  • Google Dictionary (by Google) - 谷歌字典，可双击或选中英英翻译
  • Google 翻译 - 打开 Google 翻译翻译网页所选文字。中文教程
  • 彩云小译 - 双语对照网页翻译插件
  • OneTab - 将 Chrome 打开的浏览器标签合成一个，减少浏览器内存。中文教程
  • Quick Tabs - 像 IDEA 查看 Recent Files 一样查看 Recent Tabs，全屏操作下使用更好，可用 Tab / Shift Tab 上下切换，FireFox 可 Ctrl + Tab 直接预览
  • Copy page title and url - 一个复制浏览器 URL 的插件，辅助全屏操作

油猴脚本

  • AC-baidu - 一款去除（百度）搜索引擎右侧广告，更改搜索页样式的脚本。建议用 Google :-)
  • 网盘助手 - 一款生成下载链接的的网盘助手（可用于百度网盘）

英语

  • Merriam-Webster - 美观实用的英英字典
  • 欧路词典 - 干净、简洁的词典
  • Etymonline - 一个查询单词起源、历史的词典，可查询单词起源、历史，用于词根记忆。
  • 每日英语听力 - 欧路词典出品的英语听力 APP，免费，干净，内容丰富。可切换英音、美音，可默认发音，译文开闭可调，可测评。
  • 轻听英语 - UI 特别优秀的英语听力 APP
  • VOA - 可读每日一句、每日新闻的英语学习 iOS APP，免费，大量用户打卡
  • VOA 慢速英语 - 一款可语音测评的 VOA 慢速英语 Andriod App。另有VOA常速英语
  • 翻译 - 一款 Google 出品的翻译 App，比较准，Android 版悬浮窗很实用，缺点是有的单词没音标
  • 巴别英语 - 刷视频学口语，单句重复 / 灵活字幕 / 情境填词
  • ELSA Speak英语发音学习 APP
  • 英语轻松读 - 一款直接显示生词，用于辅助英语阅读的科技新闻阅读 APP
  • Rosetta Stone - 主要以「听音辨图」的自然方法学习英语（或其他外语）的软件，收费挺贵，中文教程
  • 2 岁学说话 - 「听音识图」的英语学习 APP
  • 叽里呱啦 - 有趣的少儿英语学习 APP

绘图
